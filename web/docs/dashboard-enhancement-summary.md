## 🎯 按用户角色定制

### 举报者 (Reporter)
- 重点显示：个人贡献、记录状态、数据质量
- 隐藏：审核相关功能
- 突出：举报和查询功能

### 审核员 (Reviewer)  
- 重点显示：审核工作量、待审核队列
- 额外显示：审核历史、决策统计
- 突出：审核管理功能

### 管理员 (Admin/Super_Admin)
- 显示：完整信息
- 额外显示：系统管理入口
- 突出：管理和监控功能
